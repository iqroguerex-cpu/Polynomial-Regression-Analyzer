## 📊 Features
- Adjustable Polynomial Degree (1–10)
- R² Comparison (Linear vs Polynomial)
- Smooth curve visualization
- Underfitting & Overfitting demonstration
- Clean professional UI

---

## 🧠 What This Project Demonstrates
- Difference between linear and polynomial regression
- Model complexity impact
- Bias vs Variance concept
- Overfitting visualization
